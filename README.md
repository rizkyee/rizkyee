## About Me

Saya adalah Fullstack Web Developer dan Android Developer yang memiliki pengalaman dalam pengembangan aplikasi web, backend API, frontend modern, serta aplikasi mobile Android.

Saya terbiasa membangun aplikasi dengan Laravel, PHP, CodeIgniter 3, Node.js, Vue.js, React, JavaScript, Tailwind CSS, MySQL, PostgreSQL, Framework7, dan Android Studio.

Fokus utama saya adalah membuat aplikasi yang rapi, responsif, mudah digunakan, dan memiliki struktur kode yang jelas.
